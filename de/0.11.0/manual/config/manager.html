Der CometVisu Manager¶

Der Manager lässt über eine einfache grafische Oberfläche die verschiedensten Konfigurationsdateien verwalten (anlegen, bearbeiten, ersetzen, löschen). Außerdem können die die versteckten Konfigurationen so wie Medien-Datien (z.B. Hintergrundbilder) verwaltet werden.

Verfügbare CometVisu Konfigurationen¶

In der Tabelle werden alle verfügbaren Konfigurationen aufgelistet. Namen mit *) am Ende sind Demo Konfigurationen die geöffnet aber nicht geändert werden können.

Öffnen
Öffnet die Konfiguration und zeigt diese im Browser an.
Überprüfen
Überprüft die Validität der Konfiguration, also ob die Syntax der XML-Datei gültig ist.
Editieren
Öffnet die Konfiguration im grafischen Editor.
Text editieren
Öffnet die Konfiguration in einem Text-Editor im Browser.
Herunterladen
Lädt die Konfiguration herunter, so dass diese lokal gesichert oder bearbeitet werden kann.
Ersetzen
Lädt eine Konfiguration hoch und ersetzt die bestehende.
Löschen
Löscht die Konfiguration

Verfügbare Mediendateien¶

Die eigene Konfiguration lässt sich durch Dateien erweitern. Typische Beispiele sind Hintergrundbilder oder eigene CSS Regeln um ein Design nach persönlichen Vorlieben abzuändern.

Versteckte Konfigurationen¶

Manche Plugins benötigen Konfigurationseinstellungen die nicht an den Browser übertragen werden sollen. Dies wird im Handbuch unter Versteckte Konfigurationen genauer beschrieben.
