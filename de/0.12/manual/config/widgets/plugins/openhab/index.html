Das openHAB2 Plugin¶

Autor: Tobias Bräutigam
Verfügbar seit Version: 0.11.0

Beschreibung¶

Das openHAB Plugin verbessert das Zusammenspiel zwischen der CometVisu und dem openHAB2 Backend. Dieses Plugin bietet zur Zeit folgende Features:

  • Anzeige von Benachrichtigungen des Backend, z.B. wenn eine neue CometVisu Version bereitsteht und eine Möglichkeit dieses Update automatisch ausführen zu lassen.

Bemerkung

Dieses Plugin wird automatisch geladen, sobald das openHAB2 Backend benutzt wird. Es besteht also keine Notwendingkeit, dieses Plugin in der visu_config.xml zu aktivieren.
