Installation auf dem Raspberry PI¶

Die CometVisu braucht relativ wenig Ressourcen und Software-Pakete und läuft deshalb sehr gut auf einem Raspberry Pi

Da ein Raspberry Pi im Grunde ein normaler Computer mit Linux als Betriebssystem ist, gelten hier für die Installation die normalen, allgemeinen Installations-Informationen.

Für einen leichten Einstieg gibt es bei den Tutorials eine Serie die spezifisch auf die Einrichtung auf einem Raspberry Pi eingeht.
