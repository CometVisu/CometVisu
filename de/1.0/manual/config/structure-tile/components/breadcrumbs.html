Breadcrumbs

Autor: Tobias Bräutigam
Verfügbar seit: 2022

Die Breadcrumbs-Komponente erzeugt einen Navigationspfad zur aktuell sichtbaren Page. Sie wird typischerweise zusammen mit einem Menü im <header> oder <footer> eingesetzt und aktualisiert sich automatisch, sobald sich die aktuelle Seite ändert.

Um auf kleineren Bildschirmen vertikalen Platz zu sparen, unterstützen Breadcrumbs das Attribut hide-on-scroll="true". Dadurch wird die Breadcrumb-Leiste während des Scrollens im <main>-Bereich ausgeblendet und erscheint wieder, wenn die Scroll-Position an den Anfang zurückkehrt.

<header>
    <nav hide-on-scroll="true">
        <cv-menu model="pages"/>
    </nav>
    <cv-breadcrumbs hide-on-scroll="true"/>
</header>

Erlaubte Attribute

Element

Attribut

Name

Inhalt

Beschreibung

cv-breadcrumbs

hide-on-scroll

true oder false

Blendet dieses Element aus, wenn der Seiteninhalt gescrollt wurde.
