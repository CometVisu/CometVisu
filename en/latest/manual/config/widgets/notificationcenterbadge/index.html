The NotificationCenterBadge widget¶

Available since version: 0.11.0
Author: Tobias Bräutigam

Description¶

Shows the current number of messages in {@link cv.ui.NotificationCenter} and opens it on click.

Settings¶

For a general understanding of how the configuration files are structured and what elements and attributes are it is recommended to read this section first: Pages and structure of CometVisu.

The behaviour and appearance of the NotificationCenterBadge widget can be influenced by using certain attributes and elements. The following tables show the allowed attributes and elements and their possible values. The screenshots show, how both can be edited in the editor.

Attributes underlined by ..... are mandatory, all the others are optional and can be omitted.

Allowed attributes in the NotificationCenterBadge-element¶

Element Attribute
  Name Content Description
notificationcenterbadge align left, right or center  
flavour string Selection of a display variant. See also Flavour.
class string Add this value to the CSS class so that it can be formated by a user provided style sheet.
hide-when-empty true or false Hide this widget when the NotificationCenter is empty.

Elements in the editor

Examples¶

It is possible to manually edit the visu_config.xml and add an entry for the NotificationCenterBadge widget.

Caution

Make sure that you only use UTF-8 encoded characters by settings the encoding in your XML-editor to UTF-8 mode!

<notificationcenterbadge>
 <layout colspan="0"/>
</notificationcenterbadge>
