Breadcrumbs

Author: Tobias Bräutigam
Available since: 2022

The breadcrumbs component generates a navigation path to the currently visible page. It is typically placed in the <header> or <footer> together with a menu and updates automatically whenever the current page changes.

To save vertical space on smaller screens, breadcrumbs support the attribute hide-on-scroll="true". This hides the breadcrumb bar while the content in <main> is scrolled and shows it again when the scroll position returns to the top.

<header>
    <nav hide-on-scroll="true">
        <cv-menu model="pages"/>
    </nav>
    <cv-breadcrumbs hide-on-scroll="true"/>
</header>

Allowed attributes

Element

Attribute

Name

Content

Description

cv-breadcrumbs

hide-on-scroll

true or false

Fades out this content, if the page content has been scrolled.
