Energy entity

Available since: 2023
Author: Tobias Bräutigam

Description

The <cv-energy-entity> component displays the electrical energy of a specific device. It consists of an icon, a value in the unit kilo-watt-hour and an optional direction arrow.

The Energy-Entity is essentially a Svg text value with predefined color scheme, icon and special mapping for the value. These presets are bound to the type attribute. Optionally, a direction can be specified via the direction attribute.

There are the following Energy entity types:

type="pv"

type="battery" direction="incoming"

type="grid" direction="incoming"

type="battery" direction="outgoing"

type="grid" direction="outgoing"

type="charger"

type="consumer"

type="heatpump"
