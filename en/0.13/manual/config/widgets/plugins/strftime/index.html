The strftime plugin¶

Author: Michael Hausl [michael at hausl dot com]
Available since version: 0.8.0

Description¶

This plugins integrates formatted date and clock strings into based on strftime.

Settings¶

For a general understanding of how the configuration files are structured and what elements and attributes are it is recommended to read this section first: Pages and structure of CometVisu.

The behaviour and appearance of the strftime plugins can be influenced by using certain attributes and elements. The following tables show the allowed attributes and elements and their possible values. The screenshots show, how both can be edited in the editor.

Attributes underlined by ….. are mandatory, all the others are optional and be omitted.

Allowed attributes in the strftime-element¶

Element Attribute
  Name Content Description
strftime lang en, de or fr  
format string  
class string Add this value to the CSS class so that it can be formatted by a user provided style sheet.
