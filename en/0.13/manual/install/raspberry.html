Installation on Raspberry PI¶

The CometVisu needs relatively few resources and software packages and therefore runs very well on a Raspberry Pi

A quick and easy way to get a Raspberry Pi running as a “CometVisu server” is to use the KNX User Forum bereitgestellten Images

To bring the CometVisu included in the image up to date you simply follow the Update-Anweisungen
