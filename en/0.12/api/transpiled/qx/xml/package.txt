The <code>qx.xml<code> name space provides static wrapper functions to allow for cross-browser   XML handling.     The <code>Document<code> class allows you to create XML documents in various   ways; they return a native DOM document. The <code>Element<code> class   provides cross-browser methods related to DOM elements, like setting an XML   name space on an element. The <code>String<code> class provides XML-specific   escapeunescape methods.
