Toolbars are used to show a collection of menus or buttons that provide tools for the application; they are usually aligned and attached to the edge of the main application area. Example Here is a little example of how to use the widget. 

 var toolbar = new qx.ui.toolbar.ToolBar();
 var newButton = new qx.ui.toolbar.Button("New");
 toolbar.add(newButton);

This example demonstrates how to create a toolbar and a toolbar buttons. The button will be added to the toolbar in the last line. External Documentation Documentation of this widget in the qooxdoo manual.
